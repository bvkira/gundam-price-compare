# 鋼彈卡牌比價

> 建立日期：2026-06-21

## 簡介

比較多家店鋪的鋼彈卡牌市場價格，先以假資料展示 UI，未來可替換成真實資料來源。

## 檔案結構

```
.
├── index.html      # 主要頁面
├── css/
│   └── style.css   # 樣式
├── js/
│   └── main.js     # 腳本邏輯
├── README.md       # 本文件
└── .gitignore      # Git 忽略清單
```

## 功能

- 網格 / 列表切換，偏好會記錄在 localStorage。
- 即時搜尋卡牌名稱與系列。
- 依最低價低到高或高到低排序。
- 每張卡標示各家店價格，並以綠色凸顯最低價。

## 如何執行

直接用瀏覽器開啟 `index.html`，或用本機伺服器：

```bash
python -m http.server 8000
```

然後打開 http://localhost:8000

## 資料來源

目前使用 `js/main.js` 內的假資料（A店、B店、C店），未來可替換為 API 或爬蟲資料。
